Instagram Image Sizes in 2025

Instagram is the most dimension-sensitive platform. Getting the size wrong means your photo gets cropped and you lose followers' attention in the feed. Here is what works best:

Instagram Feed Posts

  • Square post: 1080 x 1080 pixels (1:1 ratio) — the classic Instagram format
  • Portrait post: 1080 x 1350 pixels (4:5 ratio) — recommended because it takes up more screen space in the feed, increasing engagement
  • Landscape post: 1080 x 566 pixels (1.91:1 ratio) — less common but useful for panoramic shots
  • Carousel posts: Use the same dimensions as above. All images in a carousel must have the same aspect ratio

Pro tip: The 4:5 portrait format (1080 x 1350) is the best-performing format on Instagram because it occupies the maximum vertical space in the feed. This means users spend more time looking at your post, which signals the algorithm to show it to more people.

Instagram Stories and Reels

  • Stories: 1080 x 1920 pixels (9:16 ratio, full screen vertical)
  • Reels: 1080 x 1920 pixels (9:16 ratio, same as stories)
  • Reels cover image: 1080 x 1920 pixels, but keep important text in the centre 1080 x 1350 area since the cover displays as 4:5 in the grid

Instagram Profile Picture

  • Size: 320 x 320 pixels (displays at 110 x 110 on mobile)
  • Displayed in a circular crop — keep the subject centred
  • Use a clear, high-contrast image that is recognisable even at small sizes

YouTube Image Sizes in 2025

YouTube Thumbnail

The YouTube thumbnail is arguably the most important image in all of social media. It directly affects your click-through rate and video views. Here are the specifications:

  • Size: 1280 x 720 pixels (16:9 ratio)
  • Minimum width: 640 pixels
  • Formats: JPG, GIF, or PNG
  • Max file size: 2 MB
  • Best practices: Use high-contrast text (3-5 words max), expressive faces, bright colours, and avoid clutter

YouTube Channel Banner

  • Recommended size: 2560 x 1440 pixels
  • Safe area for text: 1546 x 423 pixels (centre)
  • The banner displays differently on TV, desktop, tablet, and mobile — keep critical content in the safe area

Facebook Image Sizes in 2025

Facebook Post Image

  • Recommended: 1200 x 630 pixels (1.91:1 ratio)
  • Square images (1080 x 1080) also work well and take up more vertical space in the feed

Facebook Cover Photo

  • Desktop: 820 x 312 pixels
  • Mobile: 640 x 360 pixels
  • Best practice: Use 820 x 462 pixels and keep important content away from edges for cross-device compatibility

Facebook Event Cover

  • Size: 1920 x 1005 pixels (1.91:1 ratio)

LinkedIn Image Sizes in 2025

  • Post image: 1200 x 627 pixels (1.91:1 ratio) for link previews, or 1080 x 1080 for photo posts
  • Profile banner: 1584 x 396 pixels (4:1 ratio)
  • Profile picture: 400 x 400 pixels (circular crop)
  • Company page cover: 1128 x 191 pixels

Twitter/X Image Sizes in 2025

  • Post image: 1600 x 900 pixels (16:9 ratio) for single image tweets
  • Header / banner: 1500 x 500 pixels (3:1 ratio)
  • Profile picture: 400 x 400 pixels (circular crop)
  • Two images in a tweet: 700 x 800 each (7:8 ratio)

WhatsApp Image Sizes in 2025

  • Profile picture (DP): 500 x 500 pixels (circular crop, under 100 KB)
  • Status: 1080 x 1920 pixels (9:16 vertical)
  • Chat images: WhatsApp compresses to roughly 1600 x 1200. Pre-compress to 150 KB for best results using Free Toolbox's Image Compressor
